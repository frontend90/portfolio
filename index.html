EXPERIENCE
2018 - Start
Since 2018, he has become interested in IT, in particular, layout and website development. He studied theory through video lessons and various sites, putting it into practice. He started with studying HTML, then CSS and JS, gradually building up practical skills and technologies. At the moment I own: HTML, CSS (SASS, FLEX, GRID), BEM, GULP, adaptive layout, JQUERY, GITHUB. For development I use VS Code, I work with PS, FIGMA, ZEPLIN layouts. All works are presented in my portfolio in the form of cards, by clicking on which you can go to the site.
I continue to develop and build up knowledge ...
С 2018 года стал интересоваться IT, в частности, версткой и разработкой сайтов. Изучал теорию по видео урокам и различным сайтам, воплощая в практику. Начинал с изучения HTML, далее CSS и JS, постепенно наращивая практические навыки и технологии. На данный момент владею: HTML, CSS (SASS, FLEX, GRID), БЭМ, GULP, адаптивная верстка, JQUERY, GITHUB. Для разработки использую VS Code, работаю с макетами PS, FIGMA, ZEPLIN. Все работы представленны в моем портфолио в виде карточек, кликнув по которым, можно перейти на сайт.
Продолжаю развиваться и наращивать знания...
